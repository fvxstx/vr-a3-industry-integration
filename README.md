## 2. Sobre o Jogo

O jogo se passa em um ambiente industrial e foi desenvolvido com o objetivo de ensinar conceitos básicos de segurança do trabalho através da Realidade Virtual.

O jogador deverá passar por três fases contendo situações comuns encontradas em ambientes industriais. Em cada etapa será necessário tomar decisões corretas para avançar.

Ao concluir todas as fases, o treinamento é finalizado e o jogador chega ao final do jogo.

---

## 3. Fases do Jogo

### Fase 1 - Escolha dos EPIs

Nesta etapa, diversos objetos estarão disponíveis para interação. O objetivo é identificar e colocar dentro da cesta apenas os Equipamentos de Proteção Individual (EPIs) corretos.

#### Itens corretos

- Capacete de segurança
- Óculos de segurança
- Colete de segurança

#### Itens incorretos

- Martelo
- Prancheta

Caso um item incorreto seja colocado na cesta, a fase será reiniciada automaticamente.

### Fase 2 - Trabalho em Altura

Nesta fase, o jogador deverá escolher qual equipamento é adequado para realizar uma atividade acima de 2,95 metros.

#### Opções

- Escada retrátil
- Elevador de carga/peso

#### Resposta correta

- Elevador de carga/peso

Ao selecionar o equipamento correto, o jogador avança para a próxima etapa.

### Fase 3 - Proteção Respiratória

Na última fase, o jogador deverá escolher o equipamento adequado para proteção.

#### Opções

- Máscara de proteção
- Chapéu de festa

#### Resposta correta

- Máscara de proteção

Após a escolha correta, o treinamento é concluído e o jogador alcança o final do jogo.

## 4. O Problema: Segurança do Trabalho

A segurança do trabalho é uma área fundamental dentro das indústrias e empresas. Durante a integração de novos colaboradores, normalmente são realizados treinamentos para apresentar normas, procedimentos e equipamentos de proteção.

Esses treinamentos costumam depender da presença de profissionais responsáveis por orientar os funcionários sobre os riscos existentes e os EPIs necessários para cada atividade. Muitas vezes esse processo pode ser cansativo, repetitivo e pouco interativo.

O uso de um jogo em Realidade Virtual busca tornar esse aprendizado mais dinâmico e envolvente, permitindo que o usuário aprenda através da prática e da interação direta com os equipamentos apresentados.

Dessa forma, o treinamento se torna mais acessível, intuitivo e fácil de compreender.

## 5. Objetivo do Projeto

O principal objetivo do projeto é demonstrar como a Realidade Virtual pode ser utilizada como ferramenta de apoio para treinamentos de segurança do trabalho.

### Objetivos específicos

- Ensinar conceitos básicos sobre EPIs.
- Simular situações encontradas em ambientes industriais.
- Incentivar a tomada de decisões corretas.
- Tornar treinamentos mais interativos.
- Aplicar conhecimentos adquiridos na disciplina.
- Demonstrar aplicações práticas da Realidade Virtual.

---

## 6. Desenvolvimento

O projeto foi desenvolvido utilizando a Unity como motor gráfico principal.

Para a implementação das interações em Realidade Virtual foram utilizados recursos do XR Interaction Toolkit, permitindo que o jogador interaja diretamente com os objetos presentes no ambiente.

### Funcionalidades implementadas

- Interação com objetos.
- Sistema de fases.
- Progressão de objetivos.
- Verificação de respostas corretas.
- Reinício automático em caso de erro.
- Ambiente virtual industrial.
- Sistema de conclusão de jogo.
